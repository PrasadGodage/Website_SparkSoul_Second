Dot Net Developer

Job Titile - .Net Full Stack Developer (MVC, SQL Server)
Salary - 12000 - 18000
Experience - 1 to 2 years

Apply

Responsibilities:
  • Build and maintain web applications using .NET technologies, focusing on ASP.NET MVC.
  • Work with the team to add new features and fix issues in the applications.
  • Make sure the applications run smoothly and fix any problems that arise.
  • Optimize application performance and work with SQL Server databases.
  • Create user-friendly interfaces in collaboration with the UI/UX team.
  • Keep up with the latest trends in .NET web development.
  • Document code and development processes.
Requirements:
  • Bachelor's degree in Computer Science or a related field (or equivalent experience).
  • 1 to 2 years of experience in .NET web development with a focus on MVC.
  • Strong knowledge of C#, ASP.NET MVC, and .NET framework.
  • Proficiency in SQL Server and database design.
  • Familiarity with HTML, CSS, JavaScript, and jQuery.
  • Experience with version control systems like Git is a plus.
  • Understanding of RESTful AP ls and integration.
  • Strong problem-solving and communication skills.
  • Enthusiastic about learning and growing as a developer.
Preferred Qualifications:
  • Familiarity with client-side frameworks like Angular, React, or Vue.js.
  • Knowledge of Web API and Entity Framework in .NET.
  • Experience with cloud services like Azure or AWS.
  • Exposure to Agile development methodologies.
  • Contributions to open-source or personal coding projects in .NET.
